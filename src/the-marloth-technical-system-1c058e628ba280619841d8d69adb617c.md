# The Marloth technical system

# Subsystems

The Marloth technical system is divided into three subsystems:

[Science system](science-system-1c058e628ba28031b77dc6265b5aeeea.md)

[Wizardry system](wizardry-system-1c058e628ba280b9a9f0e23344e9ddc7.md)

[Dream system](dream-system-1c058e628ba280bea92adc9d0218c6e6.md)

Dream is equally important but more behind-the-scenes.
