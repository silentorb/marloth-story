# Zombie machine job

[Products: TWOLD](twold-e028aa0786f5449984a4f497c1d746fa.md)

A job where [The Agency](../Features/The%20Agency%2044db7ec455e14b10a9bc1da088f20f66.md) is tasked with building a zombie machine, similar to the one in [A Child’s Fairytale World](a-childs-fairytale-world-c368a02eabb74a47ab322615f2769723.md).

James refuses to work on the project.

Similar to the implied backstory of the first machine, the initial intent isn’t to resurrect children, but that’s all it can work on.

The main interest in the machine is to have “eternal life” and bring back loved ones.
