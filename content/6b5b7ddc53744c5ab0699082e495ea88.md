# Windows

I love windows.  My favorite setting is a cozy room with windows looking out upon a vast world with thick atmosphere.

# Setting

Windows are a useful metaphor for [The Gospel](The%20Gospel%207e8162c0d89f4dab8e997b54cdd01022.md).

At the same time, if possible, I want to also play around with there being external dimensions that are malevolent.   [Cosmic horror](../../Features/Cosmic%20horror%207015e32dadb042b995663e84d5b48388.md).

# James
