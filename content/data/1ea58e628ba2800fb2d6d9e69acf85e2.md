# No explanation

1. Entities that have no evident reason for existing or doing what they are doing.
2. Implicit narrative where not every event is spelled out.

Media is increasingly over-explaining everything.

Current writers think everything needs a backstory.

# Characters

In the case of characters, I suspect this is a lazy attempt at creating multi-dimensional characters.  To some degree that can succeed.

But characters don’t *need* to be multi-dimensional.  There is nothing wrong with simple, focused characters.
