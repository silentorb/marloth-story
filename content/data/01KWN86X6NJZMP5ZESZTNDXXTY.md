# Magic tech equation

> Any sufficiently advanced technology is indistinguishable from magic - Arthur C. Clarke’s third law

The law can be described as the following equation:

```jsx
Magic = Technology - Understanding
```

From that equation, two variations can be derived.

# Demystification - Technology

```jsx
Technology = Magic + Understanding
```

# Demystification - Understanding

```jsx
Understanding = Technology - Magic
```
