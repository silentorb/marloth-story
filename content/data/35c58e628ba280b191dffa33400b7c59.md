# Spectacles

Poignant crescendos of action.

The scope of each spectacle is usually contained within a single scene.

While spectacles are not exactly scenes, scenes with spectacles can be categorized as “spectacle scenes”.

## Requirements

Each spectacle must:

1. Make a point about an [Ideological contests](./3095477179ab4fd1a6977f151ff2aef8.md)
2. Involve some degree of kinetic action
3. Must be [Iconic](./10158e628ba280648510f4544487aab5.md) 
4. Must (subjectively) make the audience say, “Wow!”
    1. Or in the words of [Neo](./23058e628ba280c887e8f1e85aa43171.md), “Whoa.”
    2. Must be jaw-dropping / goose-bump inducing
