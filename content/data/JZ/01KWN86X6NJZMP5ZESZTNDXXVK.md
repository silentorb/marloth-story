# Fantasy misconceptions

A fantasy setting that includes misconceptions about fantasy features.

These misconceptions can be legends, rumors, beliefs.

# MC awareness

One of the main uses of this feature is for there to be popular misconceptions about subjects which the main characters have a more accurate understanding of.

This is particularly the case when they are subjects a main character is an expert in.

> 💡 Often these misconceptions/superstitions arise from people jumping to conclusions about subjects they know nothing about.

# Mystery

This overlaps with [[01KWN86X6MFZQAJ1V36T9592E3]], where there can be mystery involving which rumors are true or false.

# Mythology

This overlaps with [[01KWN86X6NJZMP5ZESZTNDXY5G]].

Sometimes the misconceptions are tied to religions, cultures, customs, and historical prejudices.

# Sliver of truth

Sometimes the misconceptions are not entirely wrong, but are distorted versions of the truth.
