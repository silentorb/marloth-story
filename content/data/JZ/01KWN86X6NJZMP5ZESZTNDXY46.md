# Overview

The purpose of this page is to explore whether to make Marloth's geography more precise.

# Dreamlike

Since Marloth is supposed to be [[01KWN86X6NJZMP5ZESZTNDXY70]], I've always leaned toward having nebulous geography.

[[01KWN86X6NJZMP5ZESZTNDXY6D]] frequently lampshades how whimsical the geography is.

# Game influence

But the older I get, I am increasingly fascinated by complex spatial layouts and the way different parts of the world relate to each other.

Games are a particularly strong influence in this area.

I've spent over a decade trying to master procedurally generating complex game worlds.

# Contiguous

I love continuity.

Similar to [[01KWN86X6MFZQAJ1V36T9592DD]] and [[01KWN86X6MFZQAJ1V36T9592JN]], I like spatial continuity.

There is something captivating about seamlessly transitioning from one location to another, where each location has a clear spatial relationship to the previous location.

# Grounded

As a part of [[01KWN86X6NJZMP5ZESZTNDXXZ8]], if I lean too far toward dreamlike landscape, the setting turns to vague mush.

# Dreamlike value

One question I need to ask is, even if precise spatial world layout does hurt the dreamlike quality, is that aspect of the dreamness worth it?
