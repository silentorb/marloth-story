# Environmental dangers

I want travel in Marloth to be dangerous ([Dangerous commute](../Dangerous%20commute%2015258e628ba280309ab4d3d30701427b.md)), but I haven’t put much effort into determining what the dangers would be.

# Shades

The default top candidate has been shades.

# Beasts

I would like the beasts to be supernatural and monstrous.

# Giant monsters

Really big, slow lumbering beasts.

# Undead

The first book was big on zombies, but I don’t feel like those fit where I’m going with the second book.

# Folklore horrors

I’m not sure exactly what this would be, but I would like some kind of folklorish sort of monsters.

# Abstract horrors

Sort of like Lovecraft’s “The Colour Out of Space”, maybe there could be more abstract, impersonal dangers, such as roaming miasmas.

Maybe some form of magical anomalies like in the Stalker games?

This isn’t my first choice, but could help mix things up.

# Environmental dangers

Maybe severe weather?

# Human dangers

Maybe roaming bands of thieves?

Murderers?

Rapists?

# Villains

Humanoid characters with supernatural powers.
