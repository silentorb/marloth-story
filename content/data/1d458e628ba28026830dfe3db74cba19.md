# Types

There are three types of adventure:

1. [Action](http://127.0.0.1:5173/?node=8fb15ac964a44bf3ae98688555c43d1a) ([Action and Adventure](http://127.0.0.1:5173/?node=ac9ebbbc0e9c48d2bd56bea4bda81a6f))
2. Ambient action
3. Tension

Ambient action and Tension usually go together, but Tension does not need Ambient Action, and it is theoretically possible to have Ambient Action without Tension.

For the latter case, it is potentially a part of [Cozy horror](http://127.0.0.1:5173/?node=e5cc80dc61ed4c629951cdf472b20b7a), where the raging storm outside can make the house interior feel cozier.
