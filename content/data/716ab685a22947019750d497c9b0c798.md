# Fire from the gods

Originally I had the idea that Wizards controlled magic but some new innovation made it available to the masses.

I still think I can use that idea, but I think it would be more profound if I tweak it subtly.

What if somehow wizard artifacts became more freely available, but it is alien technology.  Groups such as the agency hack the artifacts and limp them along, but don’t fully understand them and couldn’t replicate them.
