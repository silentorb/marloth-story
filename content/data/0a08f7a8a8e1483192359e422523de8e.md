# Shared crisis

# Preamble

Nothing brings people together like a common problem.

There is something fascinating about watching a major event, particularly a negative one, from various people’s perspectives.

Disaster movies and war movies particularly capitalize on this phenomena.

For some reason, most horror movies usually don’t.

# Connecting characters and threads

Shared, negative events and paradigm shifts help tie characters and threads together.

# Different perspectives

I love seeing things from different perspectives, and it’s engaging to see the same event from different angles.

# Humility

Increasing the variety of perspectives of an event can add humility.

When an event is only presented from a single subjective perspective, the scene has a stronger gravitational pull toward narcissism.
