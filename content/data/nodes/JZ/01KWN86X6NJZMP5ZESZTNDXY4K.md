# Tragic sorceress trope

One of the main problems with the story arc I’ve had so long for Nivana is it is compelling for the wrong reasons.  It makes the audience want vengeance and to see a glorified witch.

Is there a way I can still build up all of that negative energy and redirect it in a positive direction?  I’ve had a few ideas but nothing that really stuck with me.

What I don’t want to do is pull a bait-and-switch that leaves the audience dissatisfied.

# Resolution
