# Gray hero

A character that embodies the idea that when villainy has the world by the throat, your only hope is a villain not quite as bad.

It’s a fallacy.  Good begets good and bad begets bad.  Two wrongs don’t make a right.

This is a character I want to particularly contrast with James.  If I do that, it would be good to minimize the differences between the characters so the grayness is the primary distinction.  For example, if I made the character significantly older or a woman, any conflict between them could feel like it’s partially due to their differences in station.  If it is a similar young man, then it can almost feel like a “what if” alternate version of James.

Not to say that it has to be a mirror image of James.  I still want each character to be distinct.  Just that it would be beneficial if the gray hero was in similar categories as James.
