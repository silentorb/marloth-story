# Old magic methods threatened by new methods

[[44db7ec455e14b10a9bc1da088f20f66]] largely deals in innovative [[1c058e628ba280b0b2e6c395dd9daddb]].

Each innovative magic system usually displaces some older system.

Some practitioners will be eager to move to new systems.

Some practitioners will want to stay in their old systems.

Some traditional practitioners feel threatened by the new systems.

Some traditional practitioners oppose the new systems.

The opposition takes varying degrees of hostility.

# Restraint

Care needs to be taken not to overuse this idea.

Practically every job James takes can potentially have some displaced and hostile magic group.

It’s fine for this to be a pattern, but it would start feeling awkward if this was the case for *every single job*.

# Or not

Or would it?  I’m pretty sure it would, but there is a small chance it could somehow work having pitchfork mobs around every corner.

Actually, even as I say it, that sounds kind of fun…

It especially sounds fun if James and co are hounded by parties from various angles: People hating them for one reason and other people hating them for opposite reasons.
