# SSS

# Overview

SSS (or Triple S) is a system for applying wholesome drama to narrative.

1. Sin
2. Suffering
3. Sanctity

1. Identify spiritual weaknesses of the character
2. ~~Give the character tailor-made trials that touch their weaknesses~~
3. Use that to lead to sanctification in the character

~~When identifying spiritual weaknesses, look for what Earthly things the character values.~~

~~What Earthly things the character values, the character will also idolize.~~

~~What the character idolizes, they will also sin over.~~

# Backing up

I think there may still be some value here, but after thinking about it more, I don’t think trials *need* to be tailored to personalities.

Doing so can be useful from time to time, but shouldn’t be prescriptive.

Being prescriptive adds unnecessary constraints on trail matchmaking and I suspect it could start to feel contrived.
