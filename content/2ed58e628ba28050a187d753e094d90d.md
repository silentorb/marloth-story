# Marloth book style overview

While there are common themes across the Marloth setting, each Marloth book has a distinctive sub-style.

This article provides a high-level summary of the distinguishing style of each book.

Marloth in general is dark fantasy, but each book has a different kind of darkness.

# Book 1 - A Child’s Fairytale World

Prominent themes and elements:

- Childhood
- Coming-of-age
- Perversion
- Creepy
- Toys

# Book 2 - TWOLD

I always pictured the second to be the grungiest and gritty of the series.

Prominent themes and elements:

- Wilderness
- Working class
- Superstition
- Survival
- Magic
- Marriage

# Book 3 - The Shadowhood

While all three Marloth books prominently feature cities, [The Shadowhood](../Data/Products/The%20Shadowhood%204d73a389101c473ba6bee76d5b4bf0e4.md) is the most metro of the three.

Prominent themes and elements:

- Noir
- Pulp
- Neo Tokyo
- Neon
- Nightlife
- High society
- Secret identity
- Cityscapes
- Skyscrapers
- More modern / anachronistic
- Gothic
- Parenthood
